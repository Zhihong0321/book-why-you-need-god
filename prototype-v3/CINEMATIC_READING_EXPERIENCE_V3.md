# Cinematic Reading Experience V3 — Technical & Design Document

> 文字是骨骼。音乐是血液。星空是呼吸。色彩是体温。
> 读者以为自己在"理性地阅读"，但他的灵魂已经被感官通道悄悄调动了。

---

## V3 相比 V2 的核心升级

| 升级项 | V2 | V3 |
|--------|----|----|
| 星空深度 | 单层，统一大小 | **3层视差**（far/mid/near），25倍速度差 |
| 星空对焦 | 所有星同样清晰 | **景深模拟**：远=柔光，中=微柔，近=锐利 |
| 星空旋转 | 无 | **整体缓慢旋转**，每个mood不同速度/方向 |
| 星星大小 | 随机均匀 | **幂律分布**（多小星，少大星，真实星等） |
| 闪烁 | 单频正弦 | **Perlin噪声**多频叠加（自然大气湍流） |
| 亮星效果 | 无 | **光晕(Halo)** + **JWST 6点衍射纹** |
| 背景 | 纯黑 #050508 | **多色渐变** + **噪声纹理**（胶片质感） |
| 星云形状 | 圆形blob | **SVG feTurbulence 扭曲**（有机不规则） |
| 星云层数 | 3层/mood | **6层/mood**（含丝状结构+色彩分离） |
| 星云blur | 统一70px | **分层blur**（25px–140px，有锐有柔） |
| 星云呼吸 | 无 | **opacity脉动**（±8%，5–30s周期） |
| Lightspeed星 | 大星+亮 | **只有小星**（不破坏沉浸感），近=快+亮 |
| 章节过渡 | 仅lightspeed | 新增 **Big Bang Reverse**（宇宙坍缩→闪光→标题） |
| Z轴层序 | 星云在星上方 | **星在星云上方**（星不被遮挡） |
| 背景尘埃 | 无 | **~2600个亚像素点** + 视差漂移 |

## 设计原则（不变）

1. **文字保持冷峻。** 零感性词汇。不煽情。
2. **感性通过感官通道传递。** 音乐、星空、色温——走耳朵和眼睛。
3. **SUBTLE（有文字时）。** 如果读者注意到变化的瞬间，那就太快了。
4. **EXPRESSIVE（无文字时）。** 过渡区可以尽情释放视觉能量。
5. **电影手法。** 诺兰的《星际穿越》——台词克制，Hans Zimmer让你哭。

## 星空系统 V3（Starfield Engine）

### 三层深度视差

| 层 | 占比 | 速度倍率 | 亮度倍率 | 大小倍率 | 渲染方式 | 感觉 |
|----|------|---------|---------|---------|---------|------|
| **far** | 45% | 0.1x | 0.4x | 0.35x | 柔光渐变（模拟失焦） | 百万光年外的微弱光点 |
| **mid** | 37% | 0.6x | 0.85x | 0.9x | 微柔渐变 | 肉眼可见的正常星 |
| **near** | 18% | 2.5x | 1.3x | 2.0x | 锐利圆点 | 近处飞过的明亮星 |

**视差效果：** far:near 速度比 = 1:25。当星空移动时，远星几乎不动，近星快速飞过——产生真实的3D纵深感。

### 星空旋转

整个Canvas以中心为轴缓慢旋转。不同mood有不同旋转参数：

| Mood | 旋转速度 | 方向 | 感觉 |
|------|---------|------|------|
| calm | 0.00006 rad/frame (~29分钟/圈) | 顺时针 | 自然、安定 |
| wonder | 0.00012 rad/frame (~14分钟/圈) | 顺时针 | 有什么在牵引 |
| awe | 0.00005 rad/frame (~35分钟/圈) | **逆时针** | 不安、超越尺度 |
| weight | 0.00003 rad/frame (~58分钟/圈) | **逆时针** | 几乎静止、沉重 |

**Lightspeed期间旋转冻结**（你在直线前进）。

### 亮星特效

| 特效 | 触发条件 | 描述 |
|------|---------|------|
| **光晕 (Halo)** | baseRadius > 0.8 | 径向渐变柔光圈，半径=星×3.5+2px |
| **衍射纹 (Spikes)** | baseRadius > 1.5 | JWST风格6点十字光芒，长度=星×12+6px |
| **闪耀 (Flare)** | 随机触发 | 1.5–2.8秒亮度脉冲，快攻慢衰曲线 |

### 闪烁算法（Perlin噪声）

```
scintillation = noise(t×1.0) + noise(t×2.3)×0.5 + noise(t×5.1)×0.25
```
三频叠加产生不规则的大气湍流闪烁，比单频正弦自然得多。

### 背景尘埃层

- ~2600个亚像素点（0.1–0.45px）
- 极低亮度（0.015–0.095 alpha）
- 随mood方向缓慢视差漂移
- 营造银河弥漫光的底层质感

## 星云系统 V3（Aurora/Nebula）

### Z轴层序（关键变更）

```
z-index -3: 噪声纹理（胶片颗粒）
z-index -1: 星云/极光（背景）
z-index  0: 星空Canvas（前景，pointer-events: none）
```

**星在星云上方。** 星不会被星云遮挡变暗。

### 6层结构（每个mood）

| 层 | 角色 | Blur | SVG滤镜 | 描述 |
|----|------|------|---------|------|
| a1 | 主体 | 55–70px | nebula-warp-1 | 最大面积的主色调 |
| a2 | 副体 | 45–55px | nebula-warp-2 | 不同位置的第二色 |
| a3 | 边缘 | 70–90px | nebula-warp-3 | 边缘弥散 |
| a4 | 暖色口袋 | 40–50px | nebula-warp-2 | 金/橙色局部（JWST特征） |
| a5 | **丝状结构** | 25–40px | nebula-filament | 窄长条，方向性渐变 |
| a6 | 深层包络 | 110–140px | 无 | 极大面积极低opacity |

### SVG 扭曲滤镜

```html
<filter id="nebula-warp-1">
    <feTurbulence type="fractalNoise" baseFrequency="0.012" numOctaves="4"/>
    <feDisplacementMap scale="45" xChannelSelector="R" yChannelSelector="G"/>
</filter>
```

使blob边缘变成有机不规则形状，告别完美椭圆。

### 呼吸动画

所有星云层叠加 `auroraBreathe` 动画：
```css
@keyframes auroraBreathe {
    0%   { opacity: 1; }
    50%  { opacity: 0.88; }
    100% { opacity: 1; }
}
```
周期因mood而异：awe=8s（活跃），weight=25s（几乎静止）。

## 背景系统 V3

### 深空渐变（非纯黑）

```css
background: 
    radial-gradient(ellipse at 20% 30%, rgba(8, 6, 18, 1) 0%, transparent 50%),
    radial-gradient(ellipse at 80% 70%, rgba(6, 10, 16, 1) 0%, transparent 50%),
    radial-gradient(ellipse at 50% 50%, rgba(5, 5, 12, 1) 0%, transparent 70%),
    linear-gradient(180deg, #040408, #060510, #050508, #080610);
```

真实深空不是纯黑——有极微弱的色温变化。

### 噪声纹理

SVG `feTurbulence` 生成的分形噪声，opacity 3.5%，消除"数字纯色"感。

### 非对称暗角

```css
radial-gradient(ellipse at 40% 35%, ...)
```
暗角偏左上，更电影化的构图。

## 预设摄影机 V3（Camera Presets）

### `calm` — 凝视虚空

```
星空：500颗，极慢(0.04)，中心扩散

适用：结论层、前言、镜片选择
```

### `wonder` — 有什么在那里

```
星空：750颗，缓慢上升(0.055)

亮度：[0.35, 0.8]
闪耀：0.0025（偶尔）
星云：紫色6层，SVG扭曲，呼吸9s

适用：逻辑层、生活化Lens、年轻人视角
```

### `awe` — 穿越启示

```
星空：1400颗，快速中心扩散(0.1)

```

```
星空：450颗，极慢下沉(0.015)
```

## 章节过渡 A：Lightspeed（V3 改进版）

### 核心变更：只有小星

V2的问题：大星在lightspeed时膨胀，破坏沉浸感。

V3规则：
- Lightspeed额外生成的700颗星 **全部小于0.55px**
- **不生成光晕和衍射纹**
- 近层星=快+亮，远层星=慢+暗（真实视差）
- 星的大小**不增加**，只有亮度和速度变化

| 维度 | 正常状态 | Lightspeed V3 |
|------|---------|---------------|
| 额外星数量 | — | +700（全部小星） |
| 额外星大小 | — | 0.15–0.55px（无大星） |
| 速度 | 1x | 21x（近层）/ 0.1x（远层） |
| 亮度 | 正常 | 近=×3，中=×2，远=×1.2 |
| 方向 | 当前mood | 强制 center-zoom |
| 旋转 | 当前mood | **冻结**（直线前进） |
| 星云 | 当前mood | lightspeed组 |

## 章节过渡 B：Big Bang Reverse（V3 新增）

### 设计理念

**不是"超光速旅行"。是"宇宙倒带回创世"。**

用于重大章节之间（如Part之间），比lightspeed更有仪式感。

### 完整序列

```
Phase 1 "expand"   (2.5s):  800颗小星从中心爆发向外扩散
                             星飞过屏幕边缘（制造"太快了"的感觉）
                             
Phase 2 "reverse"  (2.5s):  星减速→停止→反向加速回中心
                             前15%仍在向外漂移（惯性）
                             之后平滑加速向内（power 1.5曲线，无刹车感）
                             Aurora同时淡出至黑
                             
Phase 3 "collapse" (动态):   所有星以指数衰减(×0.88/帧)坍缩至中心
                             中心出现暖白色奇点光晕
                             等待直到**所有星到达中心**才触发下一步
                             （安全超时3秒）
                             
Phase 4 "flash"    (2s):     从奇点向外渐亮（ease-in曲线）
                             峰值72% opacity（不是纯白，保护眼睛）
                             然后回落至55%（柔和灰白）
                             背景色：暖白渐变 #FFFCF5，非纯白
                             
Phase 5 "title"    (3.5s):   章节标题淡入（scale 0.95→1 + opacity 0→1）
                             背景保持55% opacity柔和灰白
                             
Phase 6 "fade-out" (2s):     整个overlay淡出至透明
                             Aurora恢复到下一章mood
                             正常星空重新出现
```

### 护眼设计

- 闪光**不是纯白**：使用暖白渐变 `rgba(255,252,245)` → `rgba(245,242,235)`
- 峰值亮度**只有72%**，且只持续约0.8秒
- 之后**回落到55%**作为标题背景
- 从全黑到峰值使用 `progress^1.8` 曲线（慢起快收），不会突然刺眼

### 正常星空处理

Big Bang动画期间，**正常星空完全隐藏**（draw函数直接return）。
只渲染bigbang专用星阵。避免"有些星不跟着动"的穿帮。

### 使用方法

```javascript
// 在JS中调用
triggerBigBangReverse('第二章', '我不是罪人')

// 参数：副标题, 主标题
triggerBigBangReverse('Part II', '为什么我不能自我救赎')
```

```html
<!-- HTML overlay结构（已内置） -->
<div class="bigbang-overlay" id="bigbangOverlay">
    <div class="bigbang-flash"></div>
    <div class="bigbang-title" id="bigbangTitle">
        <p class="bigbang-subtitle">第二章</p>
        <h1 class="bigbang-heading">我不是罪人</h1>
    </div>
</div>
```

## 情绪弧线设计指南

### 典型弧线A（大部分章节）
```
calm → wonder → awe → [LIGHTSPEED] → weight
安静开场 → 好奇展开 → 证据高潮 → 宇宙过渡 → 沉重结尾
```

### 典型弧线B（Q&A/抵抗章）
```
wonder → awe → awe → [LIGHTSPEED] → calm
直接进入 → 论证密集 → 持续高压 → 过渡 → 平静收尾
```

### 典型弧线C（过渡章/温柔章）
```
calm → calm → wonder → weight
全程安静 → 微微展开 → 沉入永恒
```

### 典型弧线D（Part之间，使用Big Bang）
```
weight → [BIG BANG REVERSE] → calm
沉重结尾 → 宇宙坍缩→闪光→新章标题 → 安静重新开始
```

## AI 章节设计指南

### 参数速查表

```javascript
// 在 starfield-v3.js 的 CAMERAS 对象中添加/修改
CAMERAS.myMood = {
    starCount: 500,           // 200–1800，星星数量
    speed: 0.04,              // 0.01–0.15，漂移速度
    direction: 'center-zoom', // center-zoom | rise | sink | drift-left
    brightnessRange: [0.3, 0.7],  // [min, max]，0.0–1.0
    flareChance: 0.001,       // 0.0001–0.01，闪耀概率
    rotationSpeed: 0.00006,   // 0.00001–0.0003，旋转速度
    rotationDir: 1            // 1=顺时针, -1=逆时针
};
```

### 设计建议

| 场景 | 建议配置 |
|------|---------|
| 开篇"上帝的沉默" | weight + rotationSpeed: 0（完全静止） |
| 启示涌来 | awe + speed: 0.12 + starCount: 1600 |
| 温柔邀请 | wonder 默认，或 rotationSpeed 稍快 |
| 从怀疑到相信 | 开始weight → 中段切wonder |
| 高潮"所有证据汇聚" | awe + flareChance: 0.008（星星到处闪） |
| 创建自定义mood | 添加到CAMERAS对象，调用 setStarfieldMood('name') |

### 过渡选择指南

| 过渡类型 | 适用场景 | 时长 |
|---------|---------|------|
| **Lightspeed** | 普通章节之间 | ~3s（加速+巡航+减速） |
| **Big Bang Reverse** | Part之间 / 重大转折 | ~13s（完整序列） |
| **直接切换mood** | 同章内情绪变化 | 渐变（LERP_SPEED控制） |

## 性能注意事项

| 项目 | 方案 | 原因 |
|------|------|------|
| 远星模糊 | 径向渐变模拟（非ctx.filter） | ctx.filter blur 极慢，渐变GPU加速 |
| 小星渲染 | fillRect（非arc） | radius<0.5时 fillRect 快3倍 |
| 星云动画 | CSS animation（非JS） | 浏览器compositor线程，不阻塞主线程 |
| 尘埃层 | 静态位置+偏移量 | 不重新生成，只移动绘制坐标 |
| 星数上限 | 1400（awe）+ 700（lightspeed）= 2100 | 60fps无压力 |

## 核心理念（不变）

> 纸质书的逻辑：所有人看同一本书，从头到尾。
> 
> 这个 Web App 的逻辑：**文字负责理性，感官负责灵魂。两条轨道同时运行，互不干扰，但共同指向同一个结论。**
> 
> 读者读到"墙那边是永恒"时——
> 如果他在白纸黑字中读到，他会说"嗯，逻辑上是这样"。
> 如果他在深空星海+下沉星辰+渐暗星云+Escape缓缓响起的环境中读到——
> 他会说"嗯，逻辑上是这样"，同时他的胸口会有一种说不清的东西在动。
> 
> **那个"说不清的东西"，就是灵魂在回应。**
> 
> 而文字从头到尾没有煽情一个字。
